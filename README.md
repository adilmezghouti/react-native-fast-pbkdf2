# react-native-fast-pbkdf2

PBKDF2 implementation

## Installation

```sh
npm install react-native-fast-pbkdf2
```

## Usage

```js
import Pbkdf2 from "react-native-fast-pbkdf2";

// ...

const result = await Pbkdf2.multiply(3, 7);
```
